## 📌 Overview
**Gulf Tourism Guide** is a responsive multimedia web application developed as part of the *Multimedia Systems* course.

The project focuses on integrating different types of digital media into a unified interactive platform, emphasizing:
- Responsive design
- Performance optimization
- Clean UI/UX structure
- Modular front-end architecture

---

## 🎯 Objectives
- Build a fully responsive tourism website
- Optimize multimedia content for performance efficiency
- Apply clean code and scalable structure principles
- Enhance user experience through interactive UI elements

---

## 🧱 Architecture
The project follows the **Separation of Concerns (SoC)** principle:

- **HTML5** → Structure and semantic layout  
- **CSS3** → Styling, layout, and responsiveness  
- **JavaScript (ES6+)** → Interactivity and dynamic behavior  

---

## 🛠 Technologies Used
- HTML5
- CSS3 (Flexbox, Grid, Responsive Design)
- JavaScript (ES6+)
- Git & GitHub
- GitHub Pages (Deployment)

---

## ✨ Features
- Fully responsive design (mobile, tablet, desktop)
- Interactive UI components
- Optimized multimedia assets
- Smooth navigation system
- Cross-browser compatibility
